# CelularesApp - Corte 3

## Descripción

Aplicación Android desarrollada en Java utilizando Android Studio.

Características:

* Inicio de sesión.
* Registro de celulares.
* Persistencia mediante API REST.
* Uso de Retrofit.
* Almacenamiento en MockAPI.
* Sensor Acelerómetro.
* Sensor de Luz.

## Requisitos

* Android Studio.
* Android SDK 24 o superior.
* Conexión a Internet.

## Instalación

1. Descargar o clonar el repositorio.
2. Abrir el proyecto en Android Studio.
3. Esperar la sincronización de Gradle.
4. Ejecutar en un emulador o dispositivo Android.

## Funcionalidades

* Login.
* Registro de celulares.
* Guardado de información en MockAPI.
* Uso de sensores del dispositivo.
